How to Install

Follow these quick guides to launch MyCloud on your machine.

Follow these steps to configure and install MyCloud on your Mac:

  1. Download the MyCloud-macOS.dmg installer disk image.
  2. Double-click the DMG file to mount it, then drag MyCloud.app into your Applications folder.
  3. Since the app is downloaded from a browser, macOS Gatekeeper may quarantine it. To bypass this, open Terminal and run:
    xattr -cr /Applications/MyCloud.app
  4. Alternatively, right-click (Control-click) the app inside your Applications folder, select Open, and confirm. Or go to System Settings > Privacy & Security and click Open Anyway.
  5. The MyCloud Setup Wizard will open. Click Browse to select the folder where you want to store your cloud backend files.
  6. Click Install & Complete Setup. The app will automatically set up the local backend database and launch.

Follow these steps to install the executable on Windows:

  1. Download the MyCloud-Windows.exe file.
  2. Note on Windows Defender: Because this is an independent application, Windows Defender may incorrectly flag it as a virus (False Positive) and block the download. To fix this:
    • If your browser blocks it, go to your Downloads list (Ctrl+J) and select Keep dangerous file.
    • If Windows Defender deletes it, open Windows Security > Virus & threat protection > Protection history, find the blocked file, and select Allow on device or Restore.
  3. Double-click the file to launch the installer.
  4. If Windows SmartScreen displays a blue warning, click More Info, then click Run Anyway.
  5. The MyCloud Setup Wizard will open. Select where you want to store your cloud backend files.
  6. Click Install & Complete Setup. The app will copy itself to your Programs directory and add a shortcut to your Start Menu / App Menu (like WhatsApp). You can launch it from your Start Menu anytime!
